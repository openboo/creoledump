Configuration Tweaks

conky

This displays the date/time and battery status in the top-left corner of every screen. It does this by piping "conky" to dzen2.

  • Uses ".conkyrc".
  • Copied and edited script "/bin/dzconky"

fluxbox startup scripts

Added to startup scripts ".fluxbox/startup":

  • dzconky&
  • fbsetbg /Pictures/cyberpunk_hq.jpg&
  • nm-applet&
  • volti&

fluxbox slit

Added dockapps to slit:

  • wmnd -i wlp3s0&
  • wmnd -i eth0&
  • wmcpuload&
  • wmmoonclock&

fluxbox key combos

Added to ".fluxbox/keys":

  • Alt+t for 'terminal'. It's a small one...
  • Alt+g makes a bigger 'terminal'. The two sizes fit together nicely.
  • Alt+w for 'web': starts firefox.
  • Alt+f for 'find': starts dmenu_run.
  • Alt+c for 'close': closes current window.
  • Alt+x for 'xkill'.
  • Ctrl+Alt+l for 'xflock4'.

Glossary of Programs

80

Prints 80 characters to test how wide the standard terminal fits.

compton

This is the window compositor that allows transparency.

  • The transparency settings are taken care of by fluxbox
    • and then starting 'compton' makes it actually work!

conky

Used with dzen2 to display some stats about the computer at all times.

dconf2

Installed only to fix an error message received from trying to run 'turtl'.
The message was:
"libgconf-2.so.4: cannot open shared object file: No such file or directory"

dmenu

An easy way to run any command without having to open a terminal. Includes an elegant auto-completion similar to tab-completion.

dosbox

An MS-DOS emulator. Mainly for playing DOS games.

dzen2

Displays any text that is piped to it, until the parent process exits.

eel

ethmon

fluxbox

A simple windows manager with easy configuration files like .fluxbox/keys and .fluxbox/startup

  • Installed new fluxbox styles

geany

A great GUI text editor for writing code.

goodnight

This locks the screen and sleeps the computer in one simple command: "slock&systemctl suspend;"

greed

A simple and colorful math-based ASCII game where you navigate a maze of numbers, and try to eat as many numbers as you can.

kate

Text editor with a preview of the entire document shrunk down to the size of a scrollbar on the right, to show you where you are in the document.

krita

A great artist program, including animation!

leafpad

Very simple text editor.

mama

Switches the keyboard to QWERTY.

memorize

Prints out a random line from a file, one character at a time.

mplayer

To play media.

myman

A simple pac-man rip-off.

nginx

A simple web server.

  • Basic set up is done.
  • http://localhost works!

notecrypt

Allows searching through an encrypted notes file created with "vim -x".

nott

A copy of notecrypt where I'm trying to remove all the excess code.

nyancat

Shows the colorful nyancat flying across your screen indefinitely, and counts how many seconds nyancat has been running.

ovpn

Initializes openvpn with the proper settings to work with the right VPN.

pickvpn

Prints out different VPN ip addresses to choose from, one at a time. Copy and paste as the option for 'ovpn'.

sl

sl stands for "steam locomotive" and it punishes you for mistyping "ls" by having an ASCII steam locomotive drive across the terminal.

tt

Opens a narrow terminal with 'wyd', 'todo', and 'log' open in vim, opens 'memorize' in a tiny terminal below, and a big main window that doesn't have 'screen' going by default - so it can easily be used to SSH and open screen on the other side.

ttt

Opens 'pickvpn' on a tiny terminal across the top of the screen, a big main window displaying 'top', and a narrow window on the right with 'screen' executed.

theconstruct

A text-based game that allows creating new rooms in a world as imaginative as it's players. Every player can create any new screen, and anything in any sentence can lead to any other sentence, in a maze-like branching story.

turtl

An encrypted notes app that allows note-sharing. Very decentralized, yet very secure.

vivaldi

A new web browser I'm trying.

  • Installed vivaldi by .deb file. dpkg -i [file];apt-get update
  • Ctrl+Shift+N starts a new private window.
  • !warning: uses Chrome base, which is widely considered a bad thing...
  • ...but it also seems to be made by a good team, and has some spiritual roots in Opera.

vlc

To play media.

xpdf

View a pdf.

youtube-dl

Download a youtube video via command-line.

zzt

An old DOS game starts up using 'dosbox'.
